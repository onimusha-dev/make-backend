# make-backend | Documentation Website

This is the professional documentation and landing page for the `make-backend` CLI, built with **Next.js 14 (App Router)** and **Tailwind CSS**.

## Features

- **Responsive Landing Page**: Premium dark-themed UI.
- **Copy-to-Clipboard**: Easy `npx` command access.
- **Modern Stack**: TypeScript, Tailwind, Lucide Icons.

## Development

```bash
cd website
npm install
npm run dev
```

The site will be available at `http://localhost:3000`.

## Deployment

The website is ready to be deployed to **Vercel** or **GitHub Pages**.

### Vercel (Recommended)

1. Push this repository to GitHub.
2. Link the repository to a new project on [Vercel](https://vercel.com).
3. Select the `website` directory as the root for the deployment.
